# Conference Center Management System

Academic project developed in Java with SQL database support for managing conference centers, conference rooms, customers, reservations and payments.

## Features
- Insert, update and manage conference centers, rooms and customers
- Create and manage reservations
- Handle payments
- View reserved and pending conference rooms
- Filter conference rooms by capacity, seat type, WiFi and price
- Manage cancelled reservations

## Technologies
- Java
- SQL Database
- Maven

## Database Setup
To run the application locally, create your own database and execute the SQL scripts located in the `database/` folder.

Then update the database connection settings in the source code with your local:
- database name
- username
- password

## Notes
This project was developed for academic purposes. Database connection settings may need to be changed before execution on another system.
